# Briefer

A python app that displays the current and forecast weather using tkinter. Is currently setup in my dorm room using my tv screen and a raspberry pi.<br>
`weather_briefer.py` is the main program.
![Demo Image](/assets/demo-pic.png)
Credit to AWC (aviationweather.gov) for the map images, DarkSky (darksky.net) for the weather API, and IconFinder (iconfinder.com) for the weather icon assets.
